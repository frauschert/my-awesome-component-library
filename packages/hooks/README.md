# @frauschert/react-hooks

A collection of 90+ React hooks for everyday use.

## Installation

```bash
npm install @frauschert/react-hooks
# or
yarn add @frauschert/react-hooks
```

## Peer Dependencies

```json
{
    "react": "^18.0.0 || ^19.0.0",
    "react-dom": "^18.0.0 || ^19.0.0"
}
```

## Usage

```ts
import {
    useDebounce,
    useLocalStorage,
    useMediaQuery,
} from '@frauschert/react-hooks'
```
